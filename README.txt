# HBG Makeup – Portafolio Web

Este repositorio contiene el código fuente del sitio web oficial del portafolio de maquillaje **HBG Makeup**. Está basado en la plantilla **"Big Picture"** de [HTML5 UP](https://html5up.net/), adaptada para mostrar imágenes, información de servicios y una sección de contacto. Es un sitio estático ideal para mostrar trabajos de maquillaje profesional de forma elegante y responsiva.

---

## 🌐 Estructura del proyecto

hbgmakeup.github.io/
├── assets/ # Archivos CSS, JavaScript y fuentes
├── images/ # Galería de imágenes del portafolio
├── index.html # Página principal del sitio
├── sendmail.php # Script para el formulario de contacto (opcional)
├── LICENSE.txt # Licencia de la plantilla HTML5 UP
└── README.txt # Documentación original del tema


### Detalles:

- **`index.html`**: Página principal del sitio, estructurada en secciones (galería, sobre mí, contacto).
- **`assets/`**: Contiene hojas de estilo, scripts y archivos necesarios para la funcionalidad y diseño del sitio.
- **`images/`**: Carpeta con todas las imágenes usadas en la galería del portafolio.
- **`sendmail.php`**: Script para enviar correos desde el formulario de contacto (requiere servidor con PHP).
- **`LICENSE.txt`**: Licencia Creative Commons Attribution 3.0 de HTML5 UP.
- **`README.txt`**: Instrucciones del desarrollador original de la plantilla.
